# plugin.audio.beradio

## Synopsis

A very basic Kodi plugin providing an interface for the Belgian streaming radio stations.


## Motivation

Simple. Sexy.


## Installation

Copy to the Kodi plugins folder (e.g. /usr/share/kodi/addons/plugin.audio.beradio)
